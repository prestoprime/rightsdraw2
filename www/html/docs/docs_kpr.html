Key Pattern Repository

Rightsdraw holds the repository of the defined Key Pattern Rights (KPR), which can be admistrated by specifically authorised Users.

The KPR administrator can here:

  • verify the content of the deployed KPRs
  • deploy a new KPR from their editing area
  • move a KPR from one bin to another one
  • withdraw aKPR from deploy

Currently two bins are provided:

  • test
  • actual
The User can select which bin to use in the editing or in check-with activities from the settings.

A Key Pattern Rights is a MCO document defining a single Permission, with a number of conditions, without specifying any IP Entity over which the action can be acted and reference to contract and contract parties.
