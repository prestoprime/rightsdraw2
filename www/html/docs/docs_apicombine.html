MCO combine Context interfaces

The MCO combine context, with prefix /cgi-bin/rightsdraw2/combine, provides the following Interfaces:

combine guiadm

Name: combine

Description Attempts to update the OWL file instance, in the User's pprights repository, according to what is stated in the given MCO OWL contract, in the User's contracts repository. In order to be effective the two OWL documents must be consistent.

Method: GET

Parameters:

  • contract - name of OWL file in the contracts User's repository
  • instance - name of OWL file in the pprights User's repository

Name:

Description Serves to the User, as html output, a form for selecting one Contract and one PPRights instance in order to request them to be combined into an update of the latter.

Method: GET

Parameters: None
