Users Administration

It is possible to have multiple users of Rightsdraw. Each user owns his/her repository of OWL files, which are not shared. As opposite to KPR which are shared among all the user-

To add or drop users it is required to have administration privileges.

Add New User

To create a new user run from the terminal:

$cd /<yourpath>/rightsdraw2
$sudo ./adduser <username>

You will be required to set a password for the new user and confirm it. In this version the user cannot modify the password by the browser.

Dropping a User

To drop a user run from the terminal:

$cd /<yourpath>/rightsdraw2
$sudo ./drop <username>

Warning: all the files of the user will be removed.

Setting User's KPR privileges

A normal user cannot edit, deploy, and withdraw KPR. It's possible to set the User's KPR privileges by running the following command from a terminal.

$cd /<yourpath>/rightsdraw2
$./setkpr <username> on|off
