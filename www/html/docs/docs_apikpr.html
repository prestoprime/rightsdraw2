KPR Context interfaces

The KPR context, with prefix /cgi-bin/rightsdraw2/<user>/kpr, provides the following Interfaces:

copy2folder deploy guiadm list view/graph view/text withdraw

Name: copy2folder

Description Copies the OWL File representing a Key Pattern Right from sourcefolder to targetfolder. The possible source and folders are any of the Key Pattern repositories, i.e. test and actual, while as source folder it is also possible to give the specific kpr User's edititng area. It doesn't overwrite an existing document, but returns 403 Forbidden.

Method: GET

Parameters:

  • instance - name of OWL file
  • sourcefolder - folder where the OWL file is
  • targetfolder - folder destination of the copy

Returned content-type: text/plain

See also: deploy

Name: deploy

Description Uploads an OWL File expected to represent a Key Pattern Right to the given folder. It doesn't overwrite an existing document, but returns 403 Forbidden.

Method: POST

Parameters:

  • instance - name of OWL file
  • folder - folder destination of the upload (test or actual)

Returned content-type: text/html

Name: guiadm

Description Serves to the User, as html output, the lists of the Key Pattern in both test and actual shared repositories, and a number of forms for verifying the content and managing them.

Method: GET

Parameters: None

Returned content-type: text/html

Name: list

Description List the names of Key Patterns deployed in the given folder

Method: GET

Parameters:

  • folder - name of Key Pattern folder (test or actual)

Returned content-type: text/plain

Name: view/graph

Description Provides a diagram of the graph of the given Key Pattern

Method: GET

Parameters:

  • folder - name of Key Pattern folder (test or actual)
  • instance - name of OWL file

Returned content-type: text/html

Name: view/text

Description Provides a derived text describing the given Key Pattern

Method: GET

Parameters:

  • folder - name of Key Pattern folder (test or actual)
  • instance - name of OWL file

Returned content-type: text/html

Name: withdraw

Description Removes the given instance of Key Pattern from the given Key Pattern deployment forlder.

Method: GET

Parameters:

  • folder - name of Key Pattern folder (test or actual)
  • instance - name of OWL file

Returned content-type: text/plain
