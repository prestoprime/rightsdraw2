MCO Context interfaces

The MCO context, with prefix /cgi-bin/rightsdraw2/<user>/config, provides the following Interfaces:

guiadm

Name:

Description If no action is requested or html output is requested, it serves the current User's settings situations and a number for forms for requesting modifications. Otherwise it can be used to set the given uservariable either to its default value or to the given newvalue or to get information about the current value and the possible available options.

Method: GET

Parameters:

  • action - if given, one among:
    • getinfo
    • getvalue
    • setdefault
    • setoption
  • html - flag for requesting the setting current situation served as html output with a a number of forms for requesting modifications
  • newvalue
  • uservariable

Returned content-type: text/html
