Introduction to RightsDraw

Rightsdraw is a set of services for creating and handling documents according to ISO/IEC 21000-21, that is MPEG21 part 21, Media Contract Ontology (MCO).

An MCO document in RightsDraw can be:

  • a Contract, related to the exploitation rights of any set of audiovisual items, with indication of the contract parties, and optionally the inclusion of the narrative contract;
  • a rights holding information set, related to the hold exploitation rights regarding a specific audiovisual item, possibly derived from several contracts.

The services are multi-user, i.e. each user owns and manages his/her MCO documents. All users share a set of key patterns rights (KPR), which can be defined in order to make some type of work easier. Only the authorized administrative users can manage the kpr area.

RightsDraw doesn't provide a service of persistent repository, but it provides the means for interfacing with such systems, such as in the case of the services of the PrestoPRIME Preservation Platform (P4).
