RightsDraw User's settings

Each User of Rightsdraw can set/modify the values of his/her setting properties.

It is also possible to set a property to its default value, which is set in the application configuration.

The properties are:

  • Related to the Diagram representing the Rights
    • colors - specifies if the diagram must be in color or not
    • size - specififies the size (in inches) of the SVG diagram
    • omitdata - specifies for which DataProperties the data value must be omitted from the diagram, normally because it can be too large.
    • datanamepos - specifies if the name of a DataProperty must be printed on the edge (arc) or within the box
    • rankdir - specifies the direction for drawing the diagram
  • Related to how to edit MCO documents
    • mcomode - specifies if the working area and mode are those of contracts or of pprights
    • mykpr - specififies which repository of KPR to use
    • owliriprefix - specififies the prefix of the ontology IRI to be used when creating new MCO documents
  • Related to interaction with P4
    • p4account - specifies the UserID for logging in with P4 services
    • p4serverurl - specififies the base URL of the P4 services
